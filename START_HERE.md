## 🎯 המשימות שלך עכשיו (בסדר):

### 1️⃣ קרא את `GETTING_STARTED.md` 📖
זה המדריך המפורט שמסביר בדיוק מה לעשות!

### 2️⃣ צור תיקיית פרויקט 📁
```bash
mkdir ai-product-workflow
cd ai-product-workflow
```

### 3️⃣ העתק את כל הקבצים לתיקייה 📋
העבר את כל 13 הקבצים שהורדת לתיקייה החדשה

### 4️⃣ הרץ את סקריפט ההקמה ⚙️
```bash
bash setup_project.sh
```

### 5️⃣ אתחל Git 🔧
```bash
git init
git add .
git commit -m "Initial setup"
```

### 6️⃣ צור Virtual Environment 🐍
```bash
python -m venv venv
source venv/bin/activate  # Mac/Linux
# או
venv\Scripts\activate  # Windows
```

### 7️⃣ התקן חבילות 📦
```bash
pip install -r requirements.txt
```

### 8️⃣ הורד Dataset 📊
```bash
python download_dataset.py
```

### 9️⃣ אמת את ה-Dataset ✅
```bash
python dataset_selector.py data/raw/dataset.csv
```

### 🔟 דווח שסיימת! 🎉
תגיד לי "סיימתי את ההתקנה!" ונעבור לשלב הבא
